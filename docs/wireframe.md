# Multi AI Workspace - 와이어프레임 (wireframe.md)

## 1. 전체 레이아웃 구조

```
+------------------------------------------------------------------+
|                        Header (48px)                              |
|  [☰ 토글]   Multi AI Workspace                    [📋 템플릿] [⚙ 설정] |
+------------+-----------------------------------------------------+
|            |                                                     |
|  Sidebar   |              Main Content Area                      |
|  (280px)   |              (나머지 영역)                            |
|            |                                                     |
|            |                                                     |
|            |                                                     |
|            |                                                     |
|            |                                                     |
|            |                                                     |
|            |                                                     |
|            |                                                     |
|            |  +-----------------------------------------------+  |
|            |  |           Input Area (고정, 80px)              |  |
|            |  +-----------------------------------------------+  |
+------------+-----------------------------------------------------+
```

- **전체 크기**: 100vw x 100vh (브라우저 전체 화면)
- **레이아웃**: CSS Grid (2열 구조)
- **헤더**: 고정 52px
- **사이드바**: 280px 고정 (토글로 접기/펼치기)
- **입력 영역**: 하단 고정

## 3. 사이드바 (Sidebar)

```
+----------------------------+
| [+ 새 대화]                 |
+----------------------------+
| 🔍 대화 검색...             |
+----------------------------+
|                            |
| 📁 프로젝트                 |
|   ├ 📂 웹개발 프로젝트  [⋮] |
|   │   ├ 💬 React 질문       |
|   │   └ 💬 CSS 문제해결     |
|   └ 📂 번역 작업       [⋮] |
|       └ 💬 영한 번역        |
|                            |
| ─────────────────────────  |
|                            |
| 💬 일반 대화                |
|   ├ 💬 일반 질문 1     [⋮] |
|   └ 💬 일반 질문 2     [⋮] |
|                            |
| ─────────────────────────  |
|                            |
| ⭐ 즐겨찾기                 |
|   └ 💬 중요한 대화     [⋮] |
|                            |
+----------------------------+
| 📤 내보내기 | 📥 가져오기    |
+----------------------------+
```

### 3.1 새 대화 버튼
- 사이드바 최상단 고정
- 클릭 시 새 대화 세션 생성
- 메인 영역 초기화

### 3.2 검색창
- 대화 제목 및 내용 실시간 검색
- 검색어 입력 시 매칭되는 대화 필터링

### 3.3 프로젝트 폴더 섹션
```
📂 폴더 이름                   [⋮]
  └ 💬 대화 제목               [⋮]
```
- 폴더 접기/펼치기 (클릭)
- `[⋮]` 메뉴: 이름 변경, 삭제
- 대화를 폴더로 드래그앤드롭 이동

### 3.4 폴더 컨텍스트 메뉴 (`[⋮]` 클릭 시)
```
+------------------+
| ✏️ 이름 변경      |
| 🗑️ 삭제          |
+------------------+
```

### 3.5 대화 컨텍스트 메뉴 (`[⋮]` 클릭 시)
```
+------------------+
| ✏️ 이름 변경      |
| ⭐ 즐겨찾기 추가   |
| 📁 폴더로 이동    |
| 🗑️ 삭제          |
+------------------+
```

### 3.6 하단 영역
- 내보내기/가져오기 버튼 고정

|  | 🟢 ChatGPT      |                                |
|  |                 |                                |

### 4.3 AI 응답 카드 상세

```
+-----------------------------------+
|  🟣 Claude              ⏱ 2.3s   |
|  -------------------------------- |
|                                   |
|  마크다운으로 렌더링된 응답 내용     |
|  - 코드 블록 지원                  |
|  - 리스트 지원                     |
|  - **굵은 텍스트** 지원             |

|  -------------------------------- |

```

| 요소 | 설명 |
|------|------|
| AI 아이콘 + 이름 | 좌상단, AI별 고유 색상 |
| 응답 시간 | 우상단, API 응답 소요 시간 |
| 응답 내용 | 마크다운 렌더링 |
| 👍👎 평가 | 사용자 평가 (토글) |
| 📋 복사 | 응답 텍스트 클립보드 복사 |

### 4.4 로딩 상태
```
+-----------------------------------+

+-----------------------------------+

- 각 AI별 독립적 로딩 애니메이션
- 먼저 응답이 온 AI부터 순차 표시

## 5. 입력 영역 (하단 고정)

```
+-----------------------------------------------------+
|  AI 선택: [✅ Claude] [✅ Gemini] [✅ ChatGPT]        |
|  +-----------------------------------------+ [전송]  |
|  | 질문을 입력하세요... (Shift+Enter 줄바꿈) |  [▶]   |
|  +-----------------------------------------+         |
+-----------------------------------------------------+
```

### 5.1 AI 선택 체크박스
- 기본: 3개 모두 선택
- 개별 해제 가능
- 최소 1개는 선택 필수

### 5.2 텍스트 입력
- 여러 줄 입력 지원 (textarea)
- Enter: 전송
- Shift+Enter: 줄바꿈
- 자동 높이 조절 (최대 4줄)

### 5.3 전송 버튼
- 입력 내용 없을 때 비활성화 (회색)
- 전송 중 로딩 스피너 표시

## 7. 모달: 프롬프트 템플릿

```
+--------------------------------------------------+
|  📋 프롬프트 템플릿                          [✕]   |
|  ------------------------------------------------ |
|                                                  |
|  카테고리: [전체 ▼]           [+ 새 템플릿]        |
|                                                  |
|  +----------------------------------------------+|
|  | 📝 코드 리뷰 요청                         [⋮] ||
|  | 다음 코드를 리뷰해주세요: {code}               ||
|  +----------------------------------------------+|
|  | 📝 번역 요청                              [⋮] ||
|  | {text}를 {language}로 번역해주세요.            ||
|  +----------------------------------------------+|
|  | 📝 요약 요청                              [⋮] ||
|  | 다음 내용을 요약해주세요: {content}             ||
|  +----------------------------------------------+|
|                                                  |
+--------------------------------------------------+
```

### 7.1 템플릿 항목 클릭 시
- 변수(`{variable}`)가 있으면 입력 폼 표시
- 완성된 프롬프트를 입력창에 삽입

### 7.2 템플릿 생성/수정 폼
```
+--------------------------------------------------+
|  📝 템플릿 편집                              [✕]  |
|  ------------------------------------------------ |
|                                                  |
|  제목:                                            |
|  +--------------------------------------------+  |
|  | 코드 리뷰 요청                              |  |
|  +--------------------------------------------+  |
|                                                  |
|  카테고리: [개발 ▼]                               |
|                                                  |
|  내용:                                            |
|  +--------------------------------------------+  |
|  | 다음 코드를 리뷰해주세요.                     |  |
|  | 언어: {language}                             |  |
|  | 코드:                                        |  |
|  | {code}                                       |  |
|  +--------------------------------------------+  |
|                                                  |
|  💡 변수는 {변수명} 형태로 작성                     |

## 12. 인터랙션 흐름

### 12.1 질문 전송 흐름
```
[입력] → [AI 선택 확인] → [전송 클릭]
  → [입력창 비활성화]
  → [선택된 AI별 로딩 카드 표시]
  → [API 병렬 호출]
  → [응답 도착 순서대로 카드 업데이트]
  → [모든 응답 완료 시 입력창 활성화]
```

### 12.2 새 대화 생성 흐름
```
[+ 새 대화 클릭]
  → [새 대화 세션 생성]
  → [사이드바 "일반 대화"에 추가]
  → [메인 영역 초기화 (빈 상태)]
  → [첫 질문 전송 시 대화 제목 자동 생성]
```

### 12.3 폴더 관리 흐름
```
[📁+ 폴더 추가 클릭]
  → [폴더 이름 입력 모달]
  → [생성 확인]
  → [사이드바 프로젝트 섹션에 추가]
```

### 12.4 대화 이동 흐름
```
[대화 항목 드래그 시작]
  → [폴더 위 드래그 시 하이라이트]
  → [폴더에 드롭]
  → [대화가 해당 폴더로 이동]
  → [사이드바 업데이트]
```
